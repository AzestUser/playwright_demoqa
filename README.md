# DemoQA Playwright Automation

## 🇺🇦 Опис
Це простий UI-автоматизаційний проєкт на Playwright + pytest для сайту DemoQA (`https://demoqa.com`), з прикладами:
- сторінки Book Store (`book_store_page`)
- логін/профіль (`login_page`, `profile_page`)
- радіо-кнопки (`radio_button_page`)

## 🇬🇧 Description
A lightweight Playwright + pytest automation project for DemoQA (`https://demoqa.com`), including:
- Book Store page object
- login/profile pages
- radio button page

---

## 🇺🇦 Структура
- `pages/` — Page Object Model (кластери сторінок)
- `tests/` — тести pytest
- `reports/` — HTML-звіти від pytest
- `pytest.ini` — конфіг pytest

## 🇬🇧 Structure
- `pages/` — Page Object Model
- `tests/` — pytest tests
- `reports/` — pytest html reports
- `pytest.ini` — pytest config
